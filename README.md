# AI-Driven Dynamic Pricing System For E-Commerce Platform
## Overview
This project implements an AI-based dynamic pricing system for e-commerce platforms.

## Features
- Demand prediction using ML
- Dynamic price optimization
- Competitor price tracking
- Analytics dashboard

## Tech Stack
- Python (Flask)
- Machine Learning
- SQLite
- HTML, CSS, JavaScript

## Run Project
pip install -r requirements.txt
cd backend
python app.py
